Conexao.java — Classe de Conexão com MySQL

Fundação
O que é essa classe?

É a única classe que sabe como se conectar ao banco. Todas as outras classes (DAO) pegam a conexão a partir dela. É como a chave da despensa — centralizada num lugar só.

Java — Conexao.java
package conexao; // ou util, ou o pacote que você criou

import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.SQLException;
import javax.swing.JOptionPane;

public class Conexao {

    // URL de conexão: jdbc:mysql://HOST:PORTA/NOME_DO_BANCO
    private static final String URL     = "jdbc:mysql://localhost:3306/sistema";
    private static final String USUARIO = "root";
    private static final String SENHA   = "root"; // ← mude para a sua senha

    // Método que retorna a conexão
    public static Connection getConexao() {
        Connection conn = null;
        try {
            conn = DriverManager.getConnection(URL, USUARIO, SENHA);
        } catch (SQLException e) {
            JOptionPane.showMessageDialog(
                null,
                "Erro ao conectar ao banco:\n" + e.getMessage(),
                "Erro de Conexão",
                JOptionPane.ERROR_MESSAGE
            );
        }
        return conn;
    }
}

Linha por linha — o que cada parte faz:

TrechoO que faz
jdbc:mysql://localhost:3306/sistemaEndereço do banco: protocolo + host + porta + nome do banco
root (usuário)Usuário do MySQL — normalmente "root" em desenvolvimento
root (senha)Senha do MySQL — coloque a senha que você definiu na instalação
DriverManager.getConnection(...)Método do Java que abre a conexão com o banco
catch (SQLException e)Captura erros de banco (banco offline, senha errada, banco não existe)
JOptionPane.showMessageDialogExibe o erro em uma janela popup do Swing para o usuário ver
Erros mais comuns na conexão:

"Communications link failure" → MySQL não está rodando. Inicie o serviço.
"Access denied for user" → Usuário ou senha errados na classe Conexao.
"Unknown database" → O banco sistema não foi criado. Execute o SQL primeiro.
"No suitable driver" → O JAR do MySQL Connector não foi adicionado ao projeto.

Teste de Conexão — Verificar se funciona

Diagnóstico

Antes de criar qualquer tela, teste se a conexão está funcionando. Isso economiza muito tempo na prova!

Crie uma classe TesteConexao.java separada e execute ela (botão direito → Run File):

Java — TesteConexao.java
package conexao; // mesmo pacote da classe Conexao

import java.sql.Connection;
import java.sql.SQLException;

public class TesteConexao {

    public static void main(String[] args) {
        System.out.println("Tentando conectar ao banco...");

        Connection conn = Conexao.getConexao();

        if (conn != null) {
            System.out.println("✅ CONEXÃO BEM-SUCEDIDA!");
            try {
                conn.close(); // sempre feche a conexão após usar
                System.out.println("Conexão fechada com sucesso.");
            } catch (SQLException e) {
                System.out.println("Erro ao fechar: " + e.getMessage());
            }
        } else {
            System.out.println("❌ FALHOU! Verifique URL, usuário e senha.");
        }
    }
}

Model — Cliente.java (Classe de Dados)

Modelo
O que é o Model?

É a classe que representa um registro da tabela. Cada campo da tabela clientes vira um atributo privado com getter e setter. O DAO usa essa classe para passar dados entre o Java e o banco.

Tabela MySQL

id INT
nome VARCHAR(100)
email VARCHAR(150)
telefone VARCHAR(20)

Classe Java

private int id;
private String nome;
private String email;
private String telefone;

Java — Cliente.java (Model)
package model;

public class Cliente {

    // Atributos privados — espelham as colunas da tabela
    private int    id;
    private String nome;
    private String email;
    private String telefone;

    // Construtor vazio (necessário para o DAO)
    public Cliente() {}

    // Construtor completo (opcional, mas útil)
    public Cliente(String nome, String email, String telefone) {
        this.nome     = nome;
        this.email    = email;
        this.telefone = telefone;
    }

    // ─── Getters e Setters ───────────────────────────────────
    public int getId()              { return id; }
    public void setId(int id)       { this.id = id; }

    public String getNome()         { return nome; }
    public void setNome(String n)   { this.nome = n; }

    public String getEmail()        { return email; }
    public void setEmail(String e)  { this.email = e; }

    public String getTelefone()          { return telefone; }
    public void setTelefone(String t)    { this.telefone = t; }

    // toString (opcional — útil para debug)
    @Override
    public String toString() {
        return "Cliente[id=" + id + ", nome=" + nome + "]";
    }
}

Declare os atributos, depois clique com o botão direito no editor → Insert Code → Getter and Setter → selecione todos → OK. O NetBeans gera tudo!

Padrão DAO — ClienteDAO.java

DAO
DAO = Data Access Object

É o padrão onde você separa todo o código SQL em uma classe dedicada. A tela Swing não sabe nada de SQL — ela só chama métodos do DAO como dao.inserir(cliente) e dao.buscarPorNome("Ana"). Isso deixa o código organizado.

Salva um objeto Cliente no banco. Usa PreparedStatement com ? para evitar SQL Injection.

Java — inserir()
public void inserir(Cliente c) {
    // ? são os parâmetros que serão preenchidos depois
    String sql = "INSERT INTO clientes (nome, email, telefone) VALUES (?, ?, ?)";

    try (Connection conn = Conexao.getConexao();
         PreparedStatement ps = conn.prepareStatement(sql)) {

        ps.setString(1, c.getNome());     // 1º ? = nome
        ps.setString(2, c.getEmail());    // 2º ? = email
        ps.setString(3, c.getTelefone()); // 3º ? = telefone

        ps.executeUpdate(); // executa o INSERT

        JOptionPane.showMessageDialog(null, "Cliente cadastrado com sucesso!");

    } catch (SQLException e) {
        JOptionPane.showMessageDialog(null,
            "Erro ao inserir: " + e.getMessage(),
            "Erro", JOptionPane.ERROR_MESSAGE);
    }
}
Por que PreparedStatement e não Statement?

Com PreparedStatement e ?, o Java trata os valores como dados (não como SQL). Isso previne SQL Injection. Sempre use PreparedStatement quando há valores de usuário!

Retorna todos os clientes do banco como uma lista Java.

Java — listarTodos()
public List<Cliente> listarTodos() {
    List<Cliente> lista = new ArrayList<>();
    String sql = "SELECT * FROM clientes ORDER BY nome";

    try (Connection conn = Conexao.getConexao();
         Statement stmt = conn.createStatement();
         ResultSet rs   = stmt.executeQuery(sql)) {

        // Percorre cada linha retornada
        while (rs.next()) {
            Cliente c = new Cliente();
            c.setId(rs.getInt("id"));
            c.setNome(rs.getString("nome"));
            c.setEmail(rs.getString("email"));
            c.setTelefone(rs.getString("telefone"));
            lista.add(c);
        }

    } catch (SQLException e) {
        JOptionPane.showMessageDialog(null,
            "Erro ao listar: " + e.getMessage(),
            "Erro", JOptionPane.ERROR_MESSAGE);
    }

    return lista;
}

Busca clientes cujo nome contenha o texto digitado. O % no LIKE é o "curinga" — %Ana% acha "Ana", "Anabela", "Banana" etc.

Java — buscarPorNome()
public List<Cliente> buscarPorNome(String nome) {
    List<Cliente> lista = new ArrayList<>();
    String sql = "SELECT * FROM clientes WHERE nome LIKE ? ORDER BY nome";

    try (Connection conn = Conexao.getConexao();
         PreparedStatement ps = conn.prepareStatement(sql)) {

        // "%" + nome + "%" = busca por conteúdo, não exato
        ps.setString(1, "%" + nome + "%");
        ResultSet rs = ps.executeQuery();

        while (rs.next()) {
            Cliente c = new Cliente();
            c.setId(rs.getInt("id"));
            c.setNome(rs.getString("nome"));
            c.setEmail(rs.getString("email"));
            c.setTelefone(rs.getString("telefone"));
            lista.add(c);
        }

    } catch (SQLException e) {
        JOptionPane.showMessageDialog(null,
            "Erro na busca: " + e.getMessage(),
            "Erro", JOptionPane.ERROR_MESSAGE);
    }

    return lista;
}

Classe completa com todos os métodos juntos para copiar direto:

Java — ClienteDAO.java (completo)
package dao;

import conexao.Conexao;
import model.Cliente;
import java.sql.*;
import java.util.ArrayList;
import java.util.List;
import javax.swing.JOptionPane;

public class ClienteDAO {

    // ── Inserir ─────────────────────────────────────────────
    public void inserir(Cliente c) {
        String sql = "INSERT INTO clientes (nome, email, telefone) VALUES (?, ?, ?)";
        try (Connection conn = Conexao.getConexao();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setString(1, c.getNome());
            ps.setString(2, c.getEmail());
            ps.setString(3, c.getTelefone());
            ps.executeUpdate();
            JOptionPane.showMessageDialog(null, "Cliente cadastrado com sucesso!");

        } catch (SQLException e) {
            JOptionPane.showMessageDialog(null,
                "Erro ao inserir: " + e.getMessage(),
                "Erro", JOptionPane.ERROR_MESSAGE);
        }
    }

    // ── Listar todos ─────────────────────────────────────────
    public List<Cliente> listarTodos() {
        List<Cliente> lista = new ArrayList<>();
        String sql = "SELECT * FROM clientes ORDER BY nome";
        try (Connection conn = Conexao.getConexao();
             Statement stmt = conn.createStatement();
             ResultSet rs   = stmt.executeQuery(sql)) {

            while (rs.next()) {
                Cliente c = new Cliente();
                c.setId(rs.getInt("id"));
                c.setNome(rs.getString("nome"));
                c.setEmail(rs.getString("email"));
                c.setTelefone(rs.getString("telefone"));
                lista.add(c);
            }

        } catch (SQLException e) {
            JOptionPane.showMessageDialog(null,
                "Erro ao listar: " + e.getMessage(),
                "Erro", JOptionPane.ERROR_MESSAGE);
        }
        return lista;
    }

    // ── Buscar por nome ──────────────────────────────────────
    public List<Cliente> buscarPorNome(String nome) {
        List<Cliente> lista = new ArrayList<>();
        String sql = "SELECT * FROM clientes WHERE nome LIKE ? ORDER BY nome";
        try (Connection conn = Conexao.getConexao();
             PreparedStatement ps = conn.prepareStatement(sql)) {

            ps.setString(1, "%" + nome + "%");
            ResultSet rs = ps.executeQuery();

            while (rs.next()) {
                Cliente c = new Cliente();
                c.setId(rs.getInt("id"));
                c.setNome(rs.getString("nome"));
                c.setEmail(rs.getString("email"));
                c.setTelefone(rs.getString("telefone"));
                lista.add(c);
            }

        } catch (SQLException e) {
            JOptionPane.showMessageDialog(null,
                "Erro na busca: " + e.getMessage(),
                "Erro", JOptionPane.ERROR_MESSAGE);
        }
        return lista;
    }
}

Java — CadastroFrame.java (completo)
package view;

import dao.ClienteDAO;
import model.Cliente;
import javax.swing.*;
import java.awt.*;

public class CadastroFrame extends JFrame {

    // ── Componentes da tela ──────────────────────────────────
    private JTextField txtNome;
    private JTextField txtEmail;
    private JTextField txtTelefone;
    private JButton    btnSalvar;
    private JButton    btnLimpar;

    // ── Construtor — monta a janela ─────────────────────────
    public CadastroFrame() {
        setTitle("Cadastro de Clientes");
        setSize(400, 230);
        setDefaultCloseOperation(JFrame.DISPOSE_ON_CLOSE); // fecha só esta janela
        setLocationRelativeTo(null); // centraliza na tela
        setResizable(false);

        initComponents(); // monta os componentes
    }

    // ── Monta os componentes na tela ─────────────────────────
    private void initComponents() {
        // Painel principal com GridLayout (linhas x colunas x espaço)
        JPanel painel = new JPanel(new GridLayout(4, 2, 8, 8));
        painel.setBorder(BorderFactory.createEmptyBorder(15, 20, 15, 20));

        // Linha 1: Nome
        painel.add(new JLabel("Nome:"));
        txtNome = new JTextField();
        painel.add(txtNome);

        // Linha 2: Email
        painel.add(new JLabel("Email:"));
        txtEmail = new JTextField();
        painel.add(txtEmail);

        // Linha 3: Telefone
        painel.add(new JLabel("Telefone:"));
        txtTelefone = new JTextField();
        painel.add(txtTelefone);

        // Linha 4: Botões
        btnSalvar = new JButton("Salvar");
        btnLimpar = new JButton("Limpar");
        painel.add(btnSalvar);
        painel.add(btnLimpar);

        add(painel); // adiciona o painel na janela

        // ── Eventos dos botões ───────────────────────────────
        btnSalvar.addActionListener(e -> salvar());
        btnLimpar.addActionListener(e -> limpar());
    }

    // ── Salvar cliente no banco ──────────────────────────────
    private void salvar() {
        String nome = txtNome.getText().trim();

        // Validação básica
        if (nome.isEmpty()) {
            JOptionPane.showMessageDialog(this,
                "O campo Nome é obrigatório!",
                "Campos obrigatórios",
                JOptionPane.WARNING_MESSAGE);
            txtNome.requestFocus(); // foca o campo vazio
            return; // interrompe o método
        }

        // Cria o objeto com os dados da tela
        Cliente c = new Cliente();
        c.setNome(nome);
        c.setEmail(txtEmail.getText().trim());
        c.setTelefone(txtTelefone.getText().trim());

        // Chama o DAO para salvar no banco
        ClienteDAO dao = new ClienteDAO();
        dao.inserir(c);

        limpar(); // limpa os campos após salvar
    }

    // ── Limpar campos ────────────────────────────────────────
    private void limpar() {
        txtNome.setText("");
        txtEmail.setText("");
        txtTelefone.setText("");
        txtNome.requestFocus(); // foca o primeiro campo
    }
}

Busca — BuscaFrame.java (Tela Swing com JTable)

Busca
JTable — a tabela de resultados

JTable exibe dados em linhas e colunas. Ela precisa de um DefaultTableModel que é como a "fonte de dados" da tabela — você adiciona e remove linhas nele, e a tabela atualiza automaticamente.

Java — BuscaFrame.java (completo)
package view;

import dao.ClienteDAO;
import model.Cliente;
import javax.swing.*;
import javax.swing.table.DefaultTableModel;
import java.awt.*;
import java.util.List;

public class BuscaFrame extends JFrame {

    // ── Componentes ──────────────────────────────────────────
    private JTextField      txtBusca;
    private JButton         btnBuscar;
    private JButton         btnListarTodos;
    private JTable          tabela;
    private DefaultTableModel modeloTabela;

    // ── Construtor ───────────────────────────────────────────
    public BuscaFrame() {
        setTitle("Busca de Clientes");
        setSize(620, 400);
        setDefaultCloseOperation(JFrame.DISPOSE_ON_CLOSE);
        setLocationRelativeTo(null);

        initComponents();
        listarTodos(); // carrega todos ao abrir a tela
    }

    // ── Monta os componentes ─────────────────────────────────
    private void initComponents() {
        // ── Painel de busca (topo) ────────────────────────────
        JPanel painelBusca = new JPanel(new FlowLayout(FlowLayout.LEFT, 8, 8));
        painelBusca.add(new JLabel("Nome:"));

        txtBusca = new JTextField(20);
        painelBusca.add(txtBusca);

        btnBuscar     = new JButton("Buscar");
        btnListarTodos = new JButton("Listar Todos");
        painelBusca.add(btnBuscar);
        painelBusca.add(btnListarTodos);

        // ── Tabela de resultados ──────────────────────────────
        // Definir as colunas da tabela
        String[] colunas = {"ID", "Nome", "Email", "Telefone"};
        modeloTabela = new DefaultTableModel(colunas, 0) {
            @Override
            public boolean isCellEditable(int row, int col) {
                return false; // impede edição direta na tabela
            }
        };
        tabela = new JTable(modeloTabela);
        tabela.getColumnModel().getColumn(0).setMaxWidth(50); // ID pequeno
        JScrollPane scroll = new JScrollPane(tabela); // barra de rolagem

        // ── Montar a janela ───────────────────────────────────
        setLayout(new BorderLayout(5, 5));
        add(painelBusca, BorderLayout.NORTH);
        add(scroll,      BorderLayout.CENTER);

        // ── Eventos ───────────────────────────────────────────
        btnBuscar.addActionListener(e -> buscar());
        btnListarTodos.addActionListener(e -> listarTodos());

        // Enter no campo de busca também dispara a busca
        txtBusca.addActionListener(e -> buscar());
    }

    // ── Buscar por nome ──────────────────────────────────────
    private void buscar() {
        String nome = txtBusca.getText().trim();

        if (nome.isEmpty()) {
            listarTodos(); // se vazio, lista todos
            return;
        }

        modeloTabela.setRowCount(0); // limpa a tabela

        ClienteDAO dao = new ClienteDAO();
        List<Cliente> lista = dao.buscarPorNome(nome);

        for (Cliente c : lista) {
            modeloTabela.addRow(new Object[]{
                c.getId(),
                c.getNome(),
                c.getEmail(),
                c.getTelefone()
            });
        }

        if (lista.isEmpty()) {
            JOptionPane.showMessageDialog(this,
                "Nenhum cliente encontrado com o nome: " + nome,
                "Sem resultados",
                JOptionPane.INFORMATION_MESSAGE);
        }
    }

    // ── Listar todos ─────────────────────────────────────────
    private void listarTodos() {
        modeloTabela.setRowCount(0); // limpa antes de recarregar

        ClienteDAO dao = new ClienteDAO();
        List<Cliente> lista = dao.listarTodos();

        for (Cliente c : lista) {
            modeloTabela.addRow(new Object[]{
                c.getId(),
                c.getNome(),
                c.getEmail(),
                c.getTelefone()
            });
        }
    }
}

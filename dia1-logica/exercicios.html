Calculadora Simples
Iniciante

Enunciado:Complete o código abaixo para criar uma calculadora que receba dois números e uma operação (+, -, *, /) e exiba o resultado.

Código Base (complete as lacunas)
Java — Complete!
double num1 = 15;
double num2 = 4;
String operacao = "+";
double resultado = 0;

if (operacao.equals(___)) {
 resultado = num1 ___ num2;
} else if (operacao.equals("-")) {
 resultado = ____;
} else if (operacao.equals("*")) {
 resultado = ____;
} else if (operacao.equals("/")) {
 if (num2 != 0) {
 resultado = ____;
 } else {
 System.out.println("Erro: ____");
 }
}

System.out.println(num1 + " " + operacao + " " + num2 + " = " + resultado);
Dica

Use .equals()para comparar Strings (não ==). Para cada operação, use o operador aritmético correspondente entre num1e num2.

Solução Completa

Java
double num1 = 15;
double num2 = 4;
String operacao = "+";
double resultado = 0;

// .equals() compara o CONTEÚDO da String (não o endereço de memória)
if (operacao.equals("+")) {
 resultado = num1 + num2; // 15 + 4 = 19
} else if (operacao.equals("-")) {
 resultado = num1 - num2; // 15 - 4 = 11
} else if (operacao.equals("*")) {
 resultado = num1 * num2; // 15 * 4 = 60
} else if (operacao.equals("/")) {
 if (num2 != 0) {
 resultado = num1 / num2; // 15 / 4 = 3.75
 } else {
 System.out.println("Erro: divisão por zero! ");
 }
}

System.out.println(num1 + " " + operacao + " " + num2 + " = " + resultado);
// Output: 15.0 + 4.0 = 19.0
Explicação linha por linha

Declaramos as variáveis com os valores a calcular

.equals()é o método certo para comparar Strings (nunca use ==com String)

Cada bloco else ifcorresponde a uma operação diferente

Verificamos divisão por zero antes de dividir (boa prática!)

Imprimimos o resultado formatado com concatenação de strings

Tabuada Completa
Loops

Enunciado:Crie um programa que gere a tabuada completa de um número (de 1 a 10) fornecido. A saída deve ser formatada como: 5 x 3 = 15

Dica

Use um loop forque vai de 1 até 10 (inclusive). O multiplicador muda a cada iteração. O resultado é numero * i.

Solução

Java
int numero = 7; // Troque para qualquer número!

System.out.println("=== Tabuada do " + numero + " ===");

// O for repete de 1 até 10
for (int i = 1; i <= 10; i++) {
 int resultado = numero * i;
 // \t é uma tabulação para alinhar a saída
 System.out.printf("%d x %2d = %3d%n", numero, i, resultado);
}

/* Saída:
=== Tabuada do 7 ===
7 x 1 = 7
7 x 2 = 14
7 x 3 = 21
...
7 x 10 = 70
*/

// DESAFIO EXTRA: Tabuada de TODOS os números (1-10)
System.out.println("\n=== Tabuada Completa ===");
for (int n = 1; n <= 10; n++) {
 System.out.print("Tabuada do " + n + ": ");
 for (int i = 1; i <= 10; i++) {
 System.out.print(n * i + " ");
 }
 System.out.println(); // quebra de linha
}

Média de Notas e Situação
Intermediário

Enunciado:Crie um programa que receba 3 notas de um aluno, calcule a média e exiba a situação: Aprovado(média ≥ 7), Recuperação(média ≥ 5) ou Reprovado(média < 5).

Dica

Some as 3 notas e divida por 3.0 (ponto flutuante!). Depois use if/else if/else com as condições de aprovação.

Solução

Java
// Notas do aluno
double nota1 = 6.5;
double nota2 = 8.0;
double nota3 = 7.5;

// Calcular média (divida por 3.0 para resultado decimal)
double media = (nota1 + nota2 + nota3) / 3.0;

// Formatar e exibir
System.out.println("=== BOLETIM ESCOLAR ===");
System.out.printf("Nota 1: %.1f%n", nota1);
System.out.printf("Nota 2: %.1f%n", nota2);
System.out.printf("Nota 3: %.1f%n", nota3);
System.out.printf("Média: %.2f%n", media);
System.out.print("Situação: ");

// Verificar situação
if (media >= 7.0) {
 System.out.println(" APROVADO!");
} else if (media >= 5.0) {
 System.out.println(" RECUPERAÇÃO");
} else {
 System.out.println(" REPROVADO");
}

/* Saída:
=== BOLETIM ESCOLAR ===
Nota 1: 6.5
Nota 2: 8.0
Nota 3: 7.5
Média: 7.33
Situação: APROVADO!
*/

Verificador de Palíndromo
Avançado

Enunciado:Verifique se uma palavra é um palíndromo (lida ao contrário é igual ao original). Exemplos: arara, radar, racecar.

Dica de lógica

Compare o primeiro caractere com o último, o segundo com o penúltimo, e assim por diante. Ou inverta a String inteira e compare com a original.

Dica

Use new StringBuilder(palavra).reverse().toString()para inverter uma String em Java. Depois compare com .equals(). Converta tudo para minúsculas com .toLowerCase().

Solução

Java
String palavra = "Arara";

// Converter para minúsculas para ignorar maiúsculas/minúsculas
String palavraNormal = palavra.toLowerCase();

// Inverter a string usando StringBuilder
String palavraInvertida = new StringBuilder(palavraNormal).reverse().toString();

System.out.println("Palavra: " + palavra);
System.out.println("Invertida: " + palavraInvertida);

if (palavraNormal.equals(palavraInvertida)) {
 System.out.println(" É um PALÍNDROMO!");
} else {
 System.out.println(" Não é palíndromo.");
}
// Saída: É um PALÍNDROMO!

// MÉTODO ALTERNATIVO (manual com loop)
static boolean ehPalindromo(String s) {
 s = s.toLowerCase();
 int inicio = 0, fim = s.length() - 1;
 while (inicio < fim) {
 if (s.charAt(inicio) != s.charAt(fim)) {
 return false;
 }
 inicio++;
 fim--;
 }
 return true;
}

Sistema de Login Simples
Prático

Enunciado:Crie um sistema de login que compare login e senha com valores fixos no código. O usuário tem 3 tentativas. Após 3 falhas, exibe "Conta bloqueada".

Dica

Use um loop forcom contador de tentativas (máximo 3). Use Scannerpara ler entrada. Quebre o loop com breakse o login for correto.

Solução Completa

Java
import java.util.Scanner;

public class SistemaLogin {
 public static void main(String[] args) {
 // Credenciais corretas (em produção, NUNCA faça isso!)
 final String LOGIN_CORRETO = "akira";
 final String SENHA_CORRETA = "java2024";

 Scanner sc = new Scanner(System.in);
 boolean logado = false;
 int maxTentativas = 3;

 System.out.println("=== SISTEMA DE LOGIN ===");

 for (int tentativa = 1; tentativa <= maxTentativas; tentativa++) {
 System.out.print("Login: ");
 String login = sc.nextLine().trim();

 System.out.print("Senha: ");
 String senha = sc.nextLine().trim();

 // Comparar com .equals() — case sensitive!
 if (login.equals(LOGIN_CORRETO) && senha.equals(SENHA_CORRETA)) {
 logado = true;
 System.out.println("\n Login realizado com sucesso!");
 System.out.println("Bem-vindo(a), " + login + "! ");
 break; // Sai do loop
 } else {
 int restantes = maxTentativas - tentativa;
 if (restantes >0) {
 System.out.println(" Login ou senha incorretos. " + restantes + " tentativa(s) restante(s).\n");
 }
 }
 }

 if (!logado) {
 System.out.println("\n Conta BLOQUEADA após 3 tentativas inválidas!");
 System.out.println("Entre em contato com o suporte.");
 }

 sc.close();
 }
}

/* Saída (com dados errados e depois certos):
=== SISTEMA DE LOGIN ===
Login: teste
Senha: 1234
 Login ou senha incorretos. 2 tentativa(s) restante(s).

Login: akira
Senha: java2024
 Login realizado com sucesso!
Bem-vindo(a), akira! 
*/

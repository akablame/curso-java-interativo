Classes e Objetos

Fundação da POO
Analogia: Forma de Bolo

Classe= a forma/moldepara fazer bolos. Define o formato.
Objeto= o bolo realfeito com a forma. Cada bolo pode ter sabor diferente (atributos diferentes), mas todos têm a mesma estrutura.

Criando sua primeira Classe

Java — Classe Pessoa
// CLASSE = molde/template
public class Pessoa {
 // ATRIBUTOS = características (o que a pessoa TEM)
 String nome;
 int idade;
 String email;

 // MÉTODO = comportamento (o que a pessoa FAZ)
 void apresentar() {
 System.out.println("Olá! Sou " + nome + ", tenho " + idade + " anos.");
 }

 void fazerAniversario() {
 idade++;
 System.out.println("Feliz aniversário, " + nome + "! Agora tem " + idade + " anos.");
 }
}

// OBJETO = instância da classe (o "bolo" feito com o molde)
public class Main {
 public static void main(String[] args) {
 // Criar objetos com "new"
 Pessoa p1 = new Pessoa();
 p1.nome = "Ana";
 p1.idade = 20;
 p1.email = "ana@email.com";

 Pessoa p2 = new Pessoa();
 p2.nome = "Carlos";
 p2.idade = 25;
 p2.email = "carlos@email.com";

 // Usar os objetos
 p1.apresentar(); // Olá! Sou Ana, tenho 20 anos.
 p2.apresentar(); // Olá! Sou Carlos, tenho 25 anos.
 p1.fazerAniversario(); // Feliz aniversário, Ana! Agora tem 21 anos.
 }
}

Atributos e Métodos

Estado e Comportamento
Analogia: Carro

Atributos= o que o carro tem: cor, marca, velocidade, combustível.
Métodos= o que o carro faz: acelerar, frear, buzinar, ligar.

Java — Classe Carro Completa
public class Carro {
 // === ATRIBUTOS (estado) ===
 String marca;
 String modelo;
 int ano;
 double velocidadeAtual; // começa em 0
 boolean ligado; // começa em false

 // === MÉTODOS (comportamento) ===
 void ligar() {
 if (!ligado) {
 ligado = true;
 System.out.println(marca + " " + modelo + ": vrum! ");
 } else {
 System.out.println("O carro já está ligado!");
 }
 }

 void desligar() {
 ligado = false;
 velocidadeAtual = 0;
 System.out.println(marca + " desligado.");
 }

 void acelerar(double km) {
 if (ligado) {
 velocidadeAtual += km;
 System.out.println("Acelerando! Velocidade: " + velocidadeAtual + " km/h");
 } else {
 System.out.println("Liga o carro primeiro! ");
 }
 }

 void frear(double km) {
 velocidadeAtual = Math.max(0, velocidadeAtual - km);
 System.out.println("Freando... Velocidade: " + velocidadeAtual + " km/h");
 }

 void buzinar() {
 System.out.println("BI-BI! ");
 }

 void exibirStatus() {
 System.out.println("=== " + marca + " " + modelo + " ===");
 System.out.println("Ano: " + ano);
 System.out.println("Ligado: " + (ligado ? "Sim " : "Não "));
 System.out.println("Velocidade: " + velocidadeAtual + " km/h");
 }
}

// Main
Carro meuCarro = new Carro();
meuCarro.marca = "Toyota";
meuCarro.modelo = "Corolla";
meuCarro.ano = 2023;

meuCarro.ligar(); // Toyota Corolla: vrum! 
meuCarro.acelerar(60); // Acelerando! Velocidade: 60.0 km/h
meuCarro.acelerar(40); // Acelerando! Velocidade: 100.0 km/h
meuCarro.buzinar(); // BI-BI! 
meuCarro.frear(30); // Freando... Velocidade: 70.0 km/h
meuCarro.exibirStatus();
Carro
+ marca: String
+ modelo: String
+ ano: int
+ velocidadeAtual: double
+ ligado: boolean
+ ligar(): void
+ desligar(): void
+ acelerar(km: double): void
+ frear(km: double): void
+ buzinar(): void

Construtores

Inicialização
Analogia: Preencher ficha no hotel

Quando você faz check-in, preenche uma ficha (nome, data, quarto). O construtor faz isso automaticamente quando um objeto é criado — define os valores iniciais obrigatórios.

Java — Construtores
public class Produto {
 String nome;
 double preco;
 int estoque;

 // CONSTRUTOR PADRÃO (sem parâmetros)
 public Produto() {
 nome = "Sem nome";
 preco = 0.0;
 estoque = 0;
 }

 // CONSTRUTOR PARAMETRIZADO (recebe valores)
 public Produto(String nome, double preco, int estoque) {
 // "this" refere-se ao atributo da classe (não ao parâmetro)
 this.nome = nome;
 this.preco = preco;
 this.estoque = estoque;
 }

 // SOBRECARGA: outro construtor (só nome e preço)
 public Produto(String nome, double preco) {
 this.nome = nome;
 this.preco = preco;
 this.estoque = 100; // estoque padrão
 }

 void exibir() {
 System.out.printf("Produto: %-15s | R$ %6.2f | Estoque: %d%n", nome, preco, estoque);
 }
}

// Usando os construtores
Produto p1 = new Produto(); // construtor padrão
Produto p2 = new Produto("Notebook", 3500.0, 10); // parametrizado
Produto p3 = new Produto("Mouse", 89.90); // só nome e preço

p1.exibir(); // Produto: Sem nome | R$ 0.00 | Estoque: 0
p2.exibir(); // Produto: Notebook | R$3500.00 | Estoque: 10
p3.exibir(); // Produto: Mouse | R$ 89.90 | Estoque: 100

Encapsulamento

Proteção de Dados
Analogia: Conta Bancária

Você não acessa o saldo diretamente — você usa o caixa eletrônico (métodos). O banco controla quem pode ver e modificar o saldo. Isso é encapsulamento: esconder o estado interno, expor apenas o necessário.

SEM encapsulamento (perigoso!)
// Qualquer um pode
// acessar diretamente!
conta.saldo = -99999; // 
conta.saldo = "abc"; // erro!
COM encapsulamento (seguro!)
// Acesso controlado!
conta.depositar(100); // OK
conta.sacar(-50); // Erro tratado
Java — ContaBancaria com Encapsulamento
public class ContaBancaria {
 // PRIVATE = só acessível dentro da classe
 private String titular;
 private double saldo;
 private String numeroConta;

 public ContaBancaria(String titular, String numero) {
 this.titular = titular;
 this.numeroConta = numero;
 this.saldo = 0.0;
 }

 // GETTER = retorna o valor (leitura)
 public String getTitular() { return titular; }
 public double getSaldo() { return saldo; }
 public String getNumeroConta() { return numeroConta; }

 // SETTER = define o valor COM validação
 public void setTitular(String titular) {
 if (titular != null && !titular.isEmpty()) {
 this.titular = titular;
 } else {
 System.out.println(" Nome inválido!");
 }
 }

 // Métodos de negócio (controlam o saldo)
 public void depositar(double valor) {
 if (valor >0) {
 saldo += valor;
 System.out.printf(" Depósito de R$ %.2f. Saldo: R$ %.2f%n", valor, saldo);
 } else {
 System.out.println(" Valor inválido para depósito!");
 }
 }

 public void sacar(double valor) {
 if (valor <= 0) {
 System.out.println(" Valor inválido!");
 } else if (valor >saldo) {
 System.out.println(" Saldo insuficiente! Saldo: R$ " + saldo);
 } else {
 saldo -= valor;
 System.out.printf(" Saque de R$ %.2f. Saldo: R$ %.2f%n", valor, saldo);
 }
 }

 public void extrato() {
 System.out.println("=== EXTRATO ===");
 System.out.println("Titular: " + titular);
 System.out.println("Conta: " + numeroConta);
 System.out.printf("Saldo: R$ %.2f%n", saldo);
 }
}

// Usando
ContaBancaria conta = new ContaBancaria("Ana Silva", "001-2345");
conta.depositar(1000); // Depósito de R$ 1000.00. Saldo: R$ 1000.00
conta.sacar(250); // Saque de R$ 250.00. Saldo: R$ 750.00
conta.sacar(1000); // Saldo insuficiente!
conta.extrato();
Regra de ouro do Encapsulamento

Use privatepara todos os atributos. Exponha via getters/setterscom validação. Isso protege o objeto de estados inválidos.

Herança

extends
Analogia: Família

Uma criança herda características dos pais (cor dos olhos, altura). Na POO, uma classe filha herda atributos e métodos da classe pai, podendo adicionar ou modificar comportamentos.

Animal
fazerSom(), mover()
↓
Cachorro
fazerSom() → "Au!"
Gato
fazerSom() → "Miau"
Pássaro
fazerSom() → "Piu"
Java — Herança Animal
// CLASSE PAI (superclasse)
public class Animal {
 protected String nome; // protected = acessível nas subclasses
 protected int idade;

 public Animal(String nome, int idade) {
 this.nome = nome;
 this.idade = idade;
 }

 public void fazerSom() {
 System.out.println(nome + " faz um som genérico...");
 }

 public void mover() {
 System.out.println(nome + " está se movendo!");
 }

 public void apresentar() {
 System.out.println("Sou " + nome + ", tenho " + idade + " anos.");
 }
}

// CLASSE FILHA (subclasse) — usa "extends"
public class Cachorro extends Animal {
 private String raca;

 public Cachorro(String nome, int idade, String raca) {
 super(nome, idade); // chama o construtor do pai!
 this.raca = raca;
 }

 // @Override = sobrescrever o método do pai
 @Override
 public void fazerSom() {
 System.out.println(nome + " diz: AU AU! ");
 }

 // Método específico do Cachorro
 public void buscarBola() {
 System.out.println(nome + " foi buscar a bolinha! ");
 }
}

public class Gato extends Animal {
 private boolean domestico;

 public Gato(String nome, int idade, boolean domestico) {
 super(nome, idade);
 this.domestico = domestico;
 }

 @Override
 public void fazerSom() {
 System.out.println(nome + " diz: MIAU ");
 }

 public void arranhar() {
 System.out.println(nome + " arranhou o sofá ");
 }
}

// Testando herança
Animal animal = new Animal("Animal genérico", 5);
Cachorro rex = new Cachorro("Rex", 3, "Labrador");
Gato mimi = new Gato("Mimi", 2, true);

animal.fazerSom(); // Animal genérico faz um som genérico...
rex.fazerSom(); // Rex diz: AU AU! 
rex.buscarBola(); // Rex foi buscar a bolinha! 
mimi.fazerSom(); // Mimi diz: MIAU 
rex.apresentar(); // Sou Rex, tenho 3 anos. (herdado do pai!)
Palavras-chave da Herança

extends— a classe filha estende a classe pai
super()— chama o construtor da classe pai
@Override— indica que estamos sobrescrevendo um método
protected— acessível na própria classe e nas subclasses

Polimorfismo

Muitas Formas
Analogia: Instrumento Musical

Todos os instrumentos tocam, mas cada um toca de forma diferente. O violão vibra cordas, a bateria bate membranas, a flauta sopra ar. Polimorfismo = mesmo método, comportamentos diferentes.

Java — Polimorfismo
// A MÁGICA do Polimorfismo: mesma lista, comportamentos diferentes!
Animal[] animais = {
 new Cachorro("Rex", 3, "Labrador"),
 new Gato("Mimi", 2, true),
 new Animal("Passarinho", 1),
 new Cachorro("Bolinha", 5, "Poodle")
};

System.out.println("=== Hora do barulho! ===");
for (Animal a : animais) {
 a.fazerSom(); // Java chama o método CORRETO para cada tipo!
}
// Saída:
// Rex diz: AU AU! 
// Mimi diz: MIAU 
// Passarinho faz um som genérico...
// Bolinha diz: AU AU! 

// instanceOf — verificar o tipo real do objeto
for (Animal a : animais) {
 if (a instanceof Cachorro) {
 Cachorro c = (Cachorro) a; // casting
 c.buscarBola();
 }
}

// INTERFACE — outro tipo de polimorfismo
public interface Voador {
 void voar(); // obriga implementar este método
}

public class Aviao implements Voador {
 @Override
 public void voar() {
 System.out.println(" Avião decolando com turbinas!");
 }
}

public class Passaro extends Animal implements Voador {
 public Passaro(String nome) { super(nome, 1); }

 @Override
 public void voar() {
 System.out.println(" " + nome + " voando com asas!");
 }

 @Override
 public void fazerSom() {
 System.out.println(nome + " diz: PUI PUI!");
 }
}
Polimorfismo na prática

Com polimorfismo, você pode criar uma lista de Animale colocar cachorros, gatos e pássaros juntos. O Java automaticamente chama o método correto de cada objeto. Isso é dispatch dinâmico!

Desafio Final POO!

Crie uma classe Funcionariocom nome e salário, e duas subclasses: Gerente(com bônus de 20%) e Desenvolvedor(com bônus de 15%). Cada um deve ter um método calcularSalarioTotal():

Java
class Funcionario {
 protected String nome;
 protected double salario;

 public Funcionario(String nome, double salario) {
 this.nome = nome;
 this.salario = salario;
 }

 public double calcularSalarioTotal() {
 return salario; // sem bônus por padrão
 }

 public void exibirContracheque() {
 System.out.printf("Funcionário: %-15s | Salário Total: R$ %.2f%n",
 nome, calcularSalarioTotal());
 }
}

class Gerente extends Funcionario {
 public Gerente(String nome, double salario) {
 super(nome, salario);
 }

 @Override
 public double calcularSalarioTotal() {
 return salario * 1.20; // 20% de bônus
 }
}

class Desenvolvedor extends Funcionario {
 public Desenvolvedor(String nome, double salario) {
 super(nome, salario);
 }

 @Override
 public double calcularSalarioTotal() {
 return salario * 1.15; // 15% de bônus
 }
}

// Main
Funcionario[] equipe = {
 new Gerente("Ana (Gerente)", 8000),
 new Desenvolvedor("Bruno (Dev)", 6000),
 new Desenvolvedor("Carla (Dev)", 5500)
};

System.out.println("=== FOLHA DE PAGAMENTO ===");
double totalFolha = 0;
for (Funcionario f : equipe) {
 f.exibirContracheque();
 totalFolha += f.calcularSalarioTotal();
}
System.out.printf("Total da folha: R$ %.2f%n", totalFolha);
